Whitespace インタプリタ

・使い方
  Java環境の整っているコマンドライン上で

> java Whitespace filename

  と実行することで、filenameを読み込み、Whitespaceのプログラムとして実行します。

・使用できる機能
  実装の関係で一部のコマンドや仕様が正式なWhitespaceのものとは異なります。
  また、本来Whitespaceに備わっている一部の機能は実装されていません。
  以下、説明のためにスペースを[SPACE]、タブを[TAB]、改行を[LF]と表記します。

基本となる構文：
[SPACE]-[コマンド] : スタック操作を行います。
[TAB]-[SPACE]-[コマンド] : スタックを引数にした演算を行います。スタックサイズが２より小さい場合は何も起きません。
[TAB]-[LF]-[コマンド] : 画面への入／出力を行います
[LF] : プログラムを終了します。
また、[SPACE]、[TAB]および[LF]以外の文字は全てコメントとみなされます。

スタック操作：
[SPACE]-[数値] : スタックの一番上に[数値]を積みます。[数値]は２進数で表現し、[SPACE]が0、[TAB]が1、そして[LF]が終端記号に相当します。
[TAB]-[SPACE]-[数値] : スタックの上から数えて[数値]番目の値をコピーしてそれをスタックの一番上に積みます。また、スタックは一番上を0番目とします。スタックサイズを超えたアクセスについては保証しません。
[LF]-[SPACE] : スタックの一番上の値を複製して新たにスタックに積みます。[TAB]-[SPACE]-[LF]と機能的には等価です。
[LF]-[TAB] : スタックの１番目と２番目の値を交換します。スタックサイズが２より小さい場合は交換に失敗します（何も起きません）
[LF]-[LF] : スタックの一番上の値を破棄します。スタックに何も積まれていない場合は何も起きません。

演算（一部のコマンドが正式なWhitespaceのものと異なるので注意）：
[SPACE]-[SPACE] : スタックの１番上と２番目の値を足し、その結果をスタックに積みます（もとの２つの値は破棄されます）。
[SPACE]-[TAB] : スタックの２番目の値からスタックの１番目の値を引いた結果をスタックに積みます。
[TAB]-[SPACE] : スタックの１番目と２番目の値の積をスタックに積みます。
[TAB]-[TAB] : スタックの２番目の値を１番目の値で割った値（切り捨て）をスタックに積みます。

標準入出力：
[SPACE]-[SPACE] : スタックの一番上の値を取り出し、それを文字として標準出力します。
[SPACE]-[TAB] : スタックの一番上の値を取り出し、それを数字として標準出力します。
[TAB]-[SPACE] : 標準入力から１文字読み込み、それを文字としてスタックに積みます。
[TAB]-[TAB] : 標準入力から１文字読み込み、それを数字（整数）としてスタックに積みます。

・サンプルプログラム
  サンプルプログラムとして、"Hello, world!"を出力するプログラム"helloworld.ws"、文字および数字を読み込んでそれを出力する"io.ws"、２つの数字を読み込み四則演算を行う"calc.ws"があるので、参考にしてください。

・実装されていない機能
  現在、以下の機能は実装されていません。
    ヒープへのアクセス
    スタックのスライド
    浮動小数演算
  これらについて今後拡張を行うかについては未定です。
  また、構文的に間違っている（例：[TAB]-[LF]-[LF]）プログラムの動作は保証していません。このときの動作についての拡張を行うかについても未定とさせていただきます。
